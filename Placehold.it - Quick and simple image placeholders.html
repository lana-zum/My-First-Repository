Welcome to placehold.it
A quick and simple image placeholder service.

How does it work? Just put your image size after our URL and you'll get a placeholder.
Like this: http://placehold.it/350x150
You can also use it in your code, like this:
<img src="http://placehold.it/350x150">
Have fun!

Image Formats

.gif, .jpeg, .jpg, .png

Adding an image file extension will render the image in the proper format. Image format is optional and the default is a gif. jpg and jpeg are the same. The image extension can go at the end of any option in the url

http://placehold.it/300.png/09f/fff
http://placehold.it/300/09f.png/fff
http://placehold.it/300/09f/fff.png

Custom Text

?text=Hello+World

Custom text can be entered using a query string at the very end of the url. This is optional, default is the image dimensions (300Ã—250). A-z (upper and lowercase), numbers, and most symbols will work just fine.

Note: Spaces become +
http://placehold.it/300?text=placehold.it+rocks!

Colors

background color / text color

Colors are represented as hex codes (#ffffff is white)

Colors always follow the dimensions
http://placehold.it/250/ffffff/000000
not
http://placehold.it/ffffff/250/000000

The first color is always the background color and the second color is the text color.

Size

width x height

Height is optional, if no height is specified the image will be a square.

Example: http://placehold.it/300
Note: Size must be the first option in the url
